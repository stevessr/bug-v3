This is a browser extension for managing custom emojis. Users can create emoji groups, organize them with tags, and quickly insert emojis on any website.

Key Features:

- Emoji management: Create groups, add emojis, organize with tags
- Quick search and favorites for frequently used emojis
- Batch import from platforms like Bilibili and Telegram
- Cloud sync via WebDAV/S3 for multi-device access
- AI-powered emoji naming (optional, requires user-provided API key)
- experiment preview UI for discourse-powered fourms

Permissions Explanation:

- storage: Saves emoji data and user settings locally
- activeTab: Injects emoji picker into current page for emoji insertion
- scripting: Dynamically injects content scripts for emoji functionality
- downloads: Allows users to export/import emoji data as files
- sidePanel: Displays emoji management interface in browser sidebar
- tabs: Manages extension popup and sidebar interactions
- notifications: Shows sync status and operation results
- host_permissions (all URLs): Required to inject emoji picker into any website

Privacy & Security:

- All emoji data is stored locally on user's device
- Cloud sync requires explicit user configuration
- No telemetry or analytics collected
- AI features require user-provided API keys
- Open source code available on GitHub
