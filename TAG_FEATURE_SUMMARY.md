## 使用方法

### 1. 編輯表情時添加標籤
1. 點擊表情的編輯按鈕（✎）
2. 在編輯模態框中找到"表情標籤"部分
3. 輸入新標籤或選擇建議標籤
4. 保存即可

### 2. 快速編輯標籤
1. 在表情網格中懸停表情
2. 點擊左下角的 🏷️ 按鈕
3. 在彈出的編輯器中管理標籤
4. 可以使用 AI 智能建議功能

### 3. 管理所有標籤
1. 進入標籤管理頁面
2. 查看、搜索、編輯或刪除標籤
3. 創建新的標籤
4. 查看標籤使用情況
